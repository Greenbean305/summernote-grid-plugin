## 📋 About

summernote-grid-plugin adds a responsive grid layout tool to the Summernote editor. Use this plugin to create columns inside your editor. It keeps your content organized so you can paste, type, or embed elements without breaking the columns. The plugin works with Bootstrap 5 and isolates content editing, making sure layout stays intact while you work.

This tool suits anyone who wants easy control over columns inside a simple text editor. No coding needed.

## 🔧 How to Use the Plugin in Summernote

If you are not a developer or lack experience with web design tools, ask someone who manages your website or editor tool to help with these steps.

1. Ensure you have Summernote editor version 0.8.18 or higher installed.
2. Download the summernote-grid-plugin files as described above.
3. Include the plugin files in your website or project folder.
4. Add the plugin script reference alongside Summernote’s core files.
5. Enable the grid plugin in your Summernote setup options.
6. Open the editor in your browser. You should see grid options in the toolbar.
7. Use the grid buttons to add, adjust, or remove columns.
8. Paste or type inside these columns. Content will stay contained within the grid.

If you want easy visual control for columns inside Summernote, this plugin keeps the layout stable while you edit.

## 🎥 Using the Grid

- Click the grid button in the editor toolbar to insert a grid layout with columns.
- Resize columns by dragging edges.
- Add or remove columns using toolbar controls.
- Paste text, images, or embed other content into each column without breaking layout.
- Columns respond when editing in contenteditable areas, isolating input so it stays inside each cell.

## 🛠️ Troubleshooting

If the plugin does not work as expected:

- Make sure your browser is updated.
- Clear browser cache and reload the editor page.
- Confirm that Bootstrap 5 CSS and JS are loaded on your page.
- If errors appear in the browser console, note their details and seek help from your site administrator or developer.
